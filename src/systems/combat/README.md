# Combat System

This directory contains the combat/battle system for the tactical RPG.

## Components

- **BattleManager**: Manages battle flow, turn order, and victory/defeat conditions
- **DamageCalculator**: Calculates damage based on stats, skills, and modifiers
- **TurnSystem**: Handles turn-based mechanics and action queue
- **CombatEffects**: Manages status effects, buffs, and debuffs
- **TargetingSystem**: Handles target selection and area of effect

## Key Features

- Turn-based tactical combat
- Grid-based positioning
- Skill and ability usage
- Status effects and conditions
- Damage types and resistances
