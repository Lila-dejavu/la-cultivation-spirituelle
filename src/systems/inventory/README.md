# Inventory System

This directory contains the inventory and equipment management system.

## Components

- **Inventory**: Item storage and management
- **Equipment**: Weapon and armor equipment system
- **ItemEffects**: Item usage and effects
- **Crafting**: Item crafting and synthesis (optional)
- **Shop**: Buy/sell system

## Key Features

- Item storage and organization
- Equipment slots (weapon, armor, accessories)
- Consumable items
- Item stacking and limits
- Equipment stat bonuses
