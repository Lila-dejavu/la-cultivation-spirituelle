# Character System

This directory contains the character management system.

## Components

- **Character**: Base character class with stats and attributes
- **CharacterStats**: HP, MP, attack, defense, speed, etc.
- **CharacterClass**: Class system (warrior, mage, etc.)
- **LevelSystem**: Experience and leveling mechanics
- **SkillTree**: Character progression and skill unlocking

## Key Features

- Character attributes and stats
- Class system with unique abilities
- Level progression
- Skill learning and mastery
- Equipment bonuses
