# Map System

This directory contains the map and grid system for tactical gameplay.

## Components

- **GridManager**: Manages the tactical grid/board
- **Pathfinding**: A* or similar algorithms for movement
- **TerrainSystem**: Different terrain types and effects
- **MovementRange**: Calculate and display movement range
- **FogOfWar**: Vision and exploration system

## Key Features

- Grid-based tactical maps
- Pathfinding for character movement
- Terrain effects (high ground, obstacles, etc.)
- Movement range visualization
- Line of sight and vision
