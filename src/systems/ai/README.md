# AI System

This directory contains the enemy AI and behavior system.

## Components

- **AIController**: Main AI decision-making
- **BehaviorTree**: AI behavior patterns
- **TacticalAI**: Tactical decision making for combat
- **Targeting**: Enemy target selection logic
- **Formations**: Enemy formation and positioning

## Key Features

- Enemy AI decision making
- Behavior patterns (aggressive, defensive, support)
- Target prioritization
- Tactical positioning
- Difficulty scaling
