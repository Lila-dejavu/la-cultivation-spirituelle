# Skill System

This directory contains the skill and ability system.

## Components

- **Skill**: Base skill class
- **SkillEffects**: Skill effects and damage calculation
- **SkillTargeting**: Target selection for skills
- **Cooldowns**: Skill cooldown management
- **SkillTree**: Skill learning and progression

## Key Features

- Active and passive skills
- Skill targeting (single, area, self)
- Skill costs (MP, AP, cooldowns)
- Skill effects (damage, healing, buffs)
- Skill combinations and chains
