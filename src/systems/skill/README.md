# 技能系統 (Skill System)

## 系統概述 (System Overview)

技能系統管理修仙世界的各種技能，包括法術、劍訣、神通、秘術和技能精通度。

The Skill System manages various skills in the cultivation world, including spells, sword arts, divine abilities, secret techniques, and skill mastery.

## 主要組件 (Main Components)

### 1. spell-system.js - 法術系統
管理元素法術和魔法。

**元素類型:**
- 五行：金、木、水、火、土
- 特殊：雷、風、冰

### 2. sword-art.js - 劍訣系統
管理劍術技能。

**劍法等級:**
- 基礎劍法 (Basic)
- 中級劍法 (Intermediate)
- 高級劍法 (Advanced)
- 劍宗絕學 (Master)
- 神劍訣 (Divine)

### 3. divine-ability.js - 神通系統
管理超自然神通能力。

### 4. secret-technique.js - 秘術系統
管理秘術和禁術。

### 5. skill-mastery.js - 技能精通度
管理技能熟練度系統。

**精通等級:**
- 初學 (Novice)
- 學徒 (Apprentice)
- 熟練 (Competent)
- 專家 (Expert)
- 大師 (Master)
- 宗師 (Grandmaster)
