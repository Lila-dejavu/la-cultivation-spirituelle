# Assets Directory

This directory contains all game assets including graphics, audio, and fonts.

## Directory Structure

### `/sprites`
Game graphics and sprites:
- Character sprites and animations
- Enemy sprites
- Tile sets and map elements
- UI elements and icons
- Effect animations

### `/audio`
Game audio files:
- Background music (BGM)
- Sound effects (SFX)
- Voice lines (optional)
- Ambient sounds

### `/fonts`
Font files for text rendering:
- UI fonts
- Dialogue fonts
- Damage number fonts

### `/effects`
Visual effects:
- Particle effects
- Skill effects
- Impact effects
- Status effect visuals

## File Organization

- Use descriptive filenames
- Organize by category and context
- Keep source files separate from game-ready assets
- Document asset requirements and specifications
