# La Cultivation Spirituelle (靈修戰棋)

A tactical RPG game focused on spiritual cultivation and strategic combat.

## 項目結構 / Project Structure

```
la-cultivation-spirituelle/
├── src/                    # 源代碼 / Source code
│   ├── core/              # 核心引擎 / Core engine
│   ├── systems/           # 遊戲系統 / Game systems
│   │   ├── combat/        # 戰鬥系統 / Combat system
│   │   ├── character/     # 角色系統 / Character system
│   │   ├── map/           # 地圖系統 / Map system
│   │   ├── inventory/     # 背包系統 / Inventory system
│   │   ├── skill/         # 技能系統 / Skill system
│   │   └── ai/            # AI系統 / AI system
│   ├── ui/                # 用戶界面 / User interface
│   │   ├── hud/           # 抬頭顯示 / HUD
│   │   ├── menu/          # 菜單 / Menus
│   │   ├── dialog/        # 對話 / Dialogs
│   │   └── battle/        # 戰鬥UI / Battle UI
│   ├── data/              # 遊戲數據 / Game data
│   │   ├── characters/    # 角色數據 / Character data
│   │   ├── skills/        # 技能數據 / Skill data
│   │   ├── items/         # 物品數據 / Item data
│   │   ├── maps/          # 地圖數據 / Map data
│   │   └── enemies/       # 敵人數據 / Enemy data
│   ├── utils/             # 工具函數 / Utilities
│   └── scenes/            # 遊戲場景 / Game scenes
├── assets/                # 遊戲資源 / Game assets
│   ├── sprites/           # 精靈圖 / Sprites
│   ├── audio/             # 音頻 / Audio
│   ├── fonts/             # 字體 / Fonts
│   └── effects/           # 特效 / Effects
└── tests/                 # 測試 / Tests
    ├── unit/              # 單元測試 / Unit tests
    └── integration/       # 整合測試 / Integration tests
```

## 主要功能 / Main Features

### 戰鬥系統 / Combat System
- 回合制戰術戰鬥 / Turn-based tactical combat
- 格子地圖定位 / Grid-based positioning
- 技能與能力 / Skills and abilities
- 狀態效果 / Status effects

### 角色系統 / Character System
- 角色屬性與數值 / Character stats and attributes
- 職業系統 / Class system
- 等級提升 / Level progression
- 技能樹 / Skill tree

### 地圖系統 / Map System
- 戰術格子地圖 / Tactical grid maps
- 尋路算法 / Pathfinding
- 地形效果 / Terrain effects
- 移動範圍顯示 / Movement range visualization
