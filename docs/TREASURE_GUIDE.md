# 寶物收集指南 (Treasure Collection Guide)

## 法寶系統 (Artifact System)

### 裝備品階
- 凡品：基礎裝備
- 靈器：靈氣灌注
- 法寶：法力凝聚
- 天器：天地認可
- 神器：神力加持
- 仙器：仙界至寶
- 混沌至寶：開天闢地

### 獲取途徑
1. 商店購買
2. 任務獎勵
3. 擊敗Boss掉落
4. 秘境探索
5. 煉器製作
6. 宗門兌換

## 丹藥系統 (Elixir System)

### 丹藥分類
- 治療丹：恢復生命
- 靈力丹：恢復靈力
- 突破丹：輔助突破
- 屬性丹：永久提升屬性
- 悟道丹：提升悟性
- 延壽丹：延長壽命

### 丹毒系統
- 服用丹藥會累積丹毒
- 丹毒過高影響效果
- 通過冥想或解毒丹清除

## 天材地寶 (Heavenly Materials)

### 材料類型
- 靈草：煉丹材料
- 礦石：煉器材料
- 妖獸內丹：煉丹煉器
- 骨骼鱗片：煉器材料
- 靈木：特殊用途
- 晶石：能量來源

### 收集方式
- 野外採集
- 妖獸掉落
- 任務獎勵
- 商店購買
- 秘境探索

## 煉器煉丹 (Crafting)

### 煉器系統
- 需要材料和配方
- 成功率受多種因素影響
- 可精煉提升品質
- 可開孔鑲嵌寶石

### 煉丹系統
- 需要丹方和材料
- 煉製過程需要控制
- 品質影響效果
- 可研究新配方

詳細實現請參考 `src/systems/treasure/` 和 `src/systems/alchemy/`
