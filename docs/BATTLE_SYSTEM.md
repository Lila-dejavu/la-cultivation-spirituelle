# 戰鬥系統說明 (Battle System Guide)

## 回合制戰棋機制 (Turn-Based Tactical Combat)

### 核心機制
- 回合制戰鬥
- 網格式地圖
- 行動順序基於速度屬性
- 陣型和站位影響戰鬥

### 戰鬥流程
1. 戰鬥開始
2. 計算行動順序
3. 角色回合
4. 敵人回合
5. 檢查勝利條件
6. 重複2-5直到戰鬥結束

## 五行相剋系統 (Five Elements System)

### 相剋關係
```
木剋土 → 土剋水 → 水剋火 → 火剋金 → 金剋木
```

### 元素反應
- 超載：雷+水/火，造成額外傷害
- 冰凍：水+冰，定身效果
- 蒸發：火+水，高額傷害
- 融化：火+冰，破除防禦

## 陣法系統 (Formation System)

### 陣法類型
- 攻擊陣：提升團隊攻擊力
- 防禦陣：提升團隊防禦力
- 輔助陣：提供各種增益效果
- 封印陣：限制敵人行動
- 幻術陣：擾亂敵人視線

### 陣法要求
- 需要多人協作
- 特定站位要求
- 境界等級限制
- 消耗靈力維持

## 技能連鎖 (Skill Combos)

### 連招系統
- 多個技能可以組合
- 特定順序觸發額外效果
- 團隊配合產生連鎖
- 元素組合產生反應

## 戰鬥策略
1. 合理利用地形
2. 注意元素相剋
3. 保持陣型完整
4. 靈活使用技能
5. 保存靈力資源
