# 故事大綱 (Story Outline)

## 世界觀設定 (World Setting)

### 修仙世界
- 三千年前，靈氣復甦
- 各大宗門林立
- 正魔兩道對立
- 秘境遍布大陸

### 時代背景
- 當前時代：靈氣復甦末期
- 天地大變將至
- 古代秘密逐漸揭開
- 飛升之門即將開啟

### 主角：林逸
