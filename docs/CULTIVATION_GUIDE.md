## 境界系統 (Realm System)

### 境界列表 (Realm List)

修仙世界共有八個主要境界，每個境界分為四個階段：

| 境界 | 英文 | 等級範圍 | 特點 |
|------|------|---------|------|
| 凝氣期 | Qi Condensation | 1-40 | 初入修仙，凝聚靈氣 |
| 築基期 | Foundation Establishment | 41-80 | 築牢根基，開拓丹田 |
| 金丹期 | Golden Core | 81-120 | 凝結金丹，壽命大增 |
| 元嬰期 | Nascent Soul | 121-160 | 孕育元嬰，神識外放 |
| 化神期 | Soul Transformation | 161-200 | 元嬰化神，法力通玄 |
| 合體期 | Body Integration | 201-240 | 神魂合一，掌控天地 |
| 渡劫期 | Tribulation Crossing | 241-280 | 渡過天劫，半步成仙 |
| 大乘期 | Mahayana | 281-320 | 大乘巔峰，準備飛升 |

### 境界階段 (Realm Stages)

每個境界包含四個階段：
1. **初期** (Early Stage) - 剛突破境界，根基未穩
2. **中期** (Middle Stage) - 境界穩固，修為增長
3. **後期** (Late Stage) - 接近圓滿，準備突破
4. **圓滿** (Peak Stage) - 境界圓滿，可嘗試突破

### 境界屬性加成 (Realm Attribute Bonuses)

每提升一個境界，角色的所有屬性都會大幅提升：

```javascript
屬性加成 = 基礎屬性 × 境界倍數 × 階段倍數

境界倍數：
- 凝氣期：1.0x
- 築基期：2.0x
- 金丹期：4.0x
- 元嬰期：8.0x
- 化神期：16.0x
- 合體期：32.0x
- 渡劫期：64.0x
- 大乘期：128.0x

階段倍數：
- 初期：1.0x
- 中期：1.25x
- 後期：1.5x
- 圓滿：1.75x
```

## 突破機制 (Breakthrough Mechanics)

### 突破條件 (Breakthrough Requirements)

要突破到下一個境界，需要滿足以下條件：

1. **境界圓滿** - 必須達到當前境界的圓滿階段
2. **靈力充足** - 累積足夠的靈力
3. **悟性要求** - 達到所需的悟性值
4. **必要資源** - 準備突破所需的丹藥和材料

### 突破成功率計算 (Success Rate Calculation)

```javascript
基礎成功率 = 50%

加成因素：
+ 靈根品質加成（最高+30%）
+ 悟性加成（最高+20%）
+ 突破丹加成（0-30%）
+ 靈脈地點加成（0-15%）
+ 師父指導加成（+15%）

最終成功率 = min(基礎成功率 + 所有加成, 95%)
```

### 突破結果 (Breakthrough Results)

1. **成功** (Success)
   - 境界提升
   - 可能觸發天劫
   - 全屬性大幅提升

2. **部分成功** (Partial Success)
   - 階段提升但境界未變
   - 靈力部分保留

3. **失敗** (Failure)
   - 損失70%靈力
   - 境界不變

4. **嚴重失敗** (Critical Failure)
   - 走火入魔
   - 損失50%靈力
   - 受到重傷（60%生命值）

### 突破準備建議 (Preparation Recommendations)

**最佳突破準備清單：**

1. 準備突破丹藥（提升成功率）
2. 尋找高品質靈脈（提供地點加成）
3. 請求師父或宗門長老指導
4. 準備療傷丹藥（以防失敗）
5. 選擇安全的時間和地點

## 靈根系統 (Spiritual Root System)

### 靈根類型 (Spiritual Root Types)

靈根決定了修煉速度和潛力：

| 類型 | 稀有度 | 修煉速度 | 特點 |
|------|--------|---------|------|
| 天靈根 | 0.1% | 5.0x | 最稀有，天賦異稟 |
| 單靈根 | 1% | 3.0x | 單一元素，純淨無比 |
| 變異靈根 | 4% | 2.5x | 特殊元素，獨特能力 |
| 雙靈根 | 10% | 2.0x | 兩種元素，平衡發展 |
| 三靈根 | 20% | 1.5x | 三種元素 |
| 四靈根 | 25% | 1.2x | 四種元素 |
| 五靈根 | 39.9% | 1.0x | 五行俱全，最常見 |

### 靈根元素 (Spiritual Root Elements)

**五行元素：**
- 金 (Metal) - 主攻擊、銳利
- 木 (Wood) - 主生命、恢復
- 水 (Water) - 主防禦、流動
- 火 (Fire) - 主爆發、燃燒
- 土 (Earth) - 主防禦、厚重

**特殊元素：**
- 雷 (Thunder) - 速度、穿透
- 風 (Wind) - 速度、範圍
- 冰 (Ice) - 控制、凍結
- 暗 (Darkness) - 隱匿、詛咒
- 光 (Light) - 淨化、治療

### 靈根品質 (Root Quality)

| 品質 | 稀有度 | 額外加成 |
|------|--------|---------|
| 神品 | 0.5% | +150%效果 |
| 天品 | 4.5% | +100%效果 |
| 上品 | 15% | +50%效果 |
| 普通 | 30% | 標準效果 |
| 凡品 | 50% | -20%效果 |

### 五行相剋 (Five Elements Cycle)

```
木剋土 → 土剋水 → 水剋火 → 火剋金 → 金剋木
Wood > Earth > Water > Fire > Metal > Wood
```

相剋效果：
- 使用相剋元素攻擊：+30%傷害
- 受到相剋元素攻擊：+30%傷害
- 相生元素：+15%效果

## 天劫系統 (Tribulation System)

### 需要渡劫的境界 (Realms Requiring Tribulation)

從元嬰期開始，每次突破都需要渡劫：

| 境界 | 劫雷波數 | 難度 |
|------|---------|------|
| 元嬰期 | 3波 | 普通 |
| 化神期 | 6波 | 困難 |
| 合體期 | 9波 | 極難 |
| 渡劫期 | 12波 | 險象環生 |
| 大乘期 | 18波（九九天劫） | 致命 |

### 天劫類型 (Tribulation Types)

1. **雷劫** (Thunder Tribulation) - 元嬰期
   - 天雷轟擊
   - 雷電麻痺

2. **心魔劫** (Heart Demon Tribulation) - 化神期
   - 內心恐懼顯化
   - 需要心境穩定

3. **元素劫** (Elemental Tribulation) - 合體期
   - 五行之力考驗
   - 全面攻擊

4. **天火劫** (Heavenly Fire Tribulation) - 渡劫期
   - 天火焚身
   - 淨化雜質

5. **九天神雷** (Nine Heaven Divine Thunder) - 大乘期
   - 最強劫雷
   - 九九八十一道

### 渡劫準備 (Tribulation Preparation)

**必備物品：**
1. 渡劫丹（提高存活率）
2. 防禦法寶
3. 療傷丹藥
4. 護身符篆

**最佳地點：**
- 高品質靈脈
- 宗門守護陣法內
- 遠離人煙（避免波及他人）

**注意事項：**
- 不可他人幫助（天劫會加強）
- 準備至少30天
- 調整狀態至最佳
- 告知師門以防萬一

### 渡劫獎勵 (Tribulation Rewards)

成功渡劫後：

1. **境界提升** - 突破到新境界
2. **劫雷淬體** - 永久屬性提升
3. **壽命延長** - 根據境界延長壽命
4. **稱號獲得** - 獲得專屬稱號
5. **經驗獎勵** - 大量修煉經驗
